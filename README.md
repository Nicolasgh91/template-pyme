# Template pyme — demo escalatunegocioconia

Plantilla de landing page para pymes argentinas, diseñada como demostración pública en el ecosistema de [escalatunegocioconia.com](https://escalatunegocioconia.com).

La misma base de código soporta múltiples demos con copy e identidad visual distintos, lo que ilustra la capacidad de reutilización del template.

## Descripción

SPA orientada a pymes que necesitan una presencia web profesional sin infraestructura compleja. Cada demo incluye: hero con propuesta de valor, grilla de productos o servicios, sección de testimonios, franja de logotipos/certificaciones y CTA final con contacto por WhatsApp o email.

El template está estructurado para que el copy, las imágenes y los colores de marca sean los únicos elementos a reemplazar entre una industria y otra.

## Stack técnico

| Área | Tecnología |
|---|---|
| Build | Vite · TypeScript |
| UI | React |
| Estilos | CSS (módulos / clases utilitarias) |
| Deploy | Vercel |

---

## Funcionalidades

- Hero con headline, subtítulo, imagen y CTAs configurables
- Grilla de productos o servicios con categorías y descripciones
- Sección de testimonios con nombre, cargo y empresa
- Franja de logos/certificaciones (SENASA, AFIP, CAME, etc.)
- CTA de cierre con botón de WhatsApp y email
- Navegación con header fijo y footer completo
- Diseño responsivo
